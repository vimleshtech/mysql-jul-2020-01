DBMS and RDBMS
=========================== 
DBMS: Database management system where we can store data in tabular format 
	Example:
		MS Excel
		MS Access
		CSV
RDBMS: Relataional database management system
     : RDBMS store data on client server architecutre
       - our data can be stored on server(one central machine) and data can be access multiple client(independent machine)
     Example:
		MY SQL Server
		MS SQL Server
		Oracle
		IBM DB2
		PostgresSQL
		MaridDB
		.....

-SQL Commands and Type
==================================
show databases;		--show list of existing database 
create database <dbname>; -- create new database
use <dbname>;		--switch database
show tables;		--show list of existing tables

Note:
	-SQL satement is not case sensetive
	-SQL statement should be terminated or end with ; (semicolon) in mysql 
	-Press enter to execute the command after ; 
	-NULL is default value in sql 
Commands and Type
=====================
DDL: data definatino language
   	- create		: ceate new objects (table, view, function , database...etc)
 	- alter			: modify existing object structure 
	- drop			: remove the object 
	- truncate 		: remove all rows from table but structure will remail same 
DML: data manipulation language	
	- insert		: save new row
	- update		: modify data in existing row
	- delete		: remove row 
DCL: data control langauge
	- create new user	
	- grant access		: add access for user
	- revoke access		: remove access for user
TCL: transaction control language
	- begin transaction	: start recording 
	- commit		: save data 
	- rollback		: cancel the changes 
DQL: data query language
	- select 		: view data 

	
Create table:
	create table <tblname> 
		(eid int, 
		fname varchar(40), 
		lname varchar(40), 
		gender varchar(1), 
		bsal int)

Show table details:
	desc <tblname>;	

Save row:
	insert into <tblname>(eid,fname,lname,gender) values(1,'raman','sinha','m');
	insert into <tblname>(eid,fname,lname,gender,bsal) values(2,'rahul','sharma','m');


Show data:
	select * from <tblname>;  -- * show all columns
	select eid,fname,lname from <tblname>; --show selected columns

	
Update data in existing row:
	
	update <tblname> set lname ='new value' where eid = val;
	update <tblname> set lname ='new value', bsal=new_val where eid = val;
	
Delete exiting selected rows:
	delete from <tblname> where eid=20;  --delete selected rows 
	delete from <tblname>;  -- delete all rows


Alter: Add new column in existing table
 	alter table <tblname> add country varchar(30);

Alter: Modify data type and size
	alter table <tblname> modify country varchar(100);

Alter: Drop existing column
	alter table <tblname> drop column country;
	
Drop: Delete table structure with data 	
	drop table <tblname>;

Truncate: Remove all rows from table but structure will remain same	
	trunate table <tblname>;
	
	
	
-Primary key:
====================================
-Primary contains or allow unique value
-Not allow null value
-Value will be physically sorted 
	1
	5
	10		

create table <tblname>(eid int primary key, fname varchar(10), lname varchar(10));

-Auto increment Keys
=====================================
-Value will be inserted automatically in sequenctial order

create table <tblname>(eid int  auto_increment primary key, fname varchar(10), lname varchar(10));

-Where Clause: is conditional statement which can be used to filter the rows
====================================
select * from <tblname> where condition;

-Operator
====================================
Conditional Operator
	=
	>
	>=
	<
	<=	
	!=			not equal	
	in
	not in
	between
	not between
	is null
	is not null 
Relational
	and	: merge two or more than two condition , return value if both(all) condition will match
	or	: merge two or more than two condition , return value either(any one or both) condition will match
Wild card: pattern match 
	like
		%	: any char and any length after or before or contains 
			'a%' any char and any length after a 

		_	: any char and fix length after or before or contains 
			'a___' any three chars after a 	

Arithmetic
	+
	-
	*
	/
	%

-Case when
===================================
	select eid,name, case when gender='m' then 'male' else 'female' from employee;
	
-Order by: arrange data in asc or desc order
==================================
	select * from employee order by eid;	--defaut is in asc
	select * from employee order by eid desc;
	select * from employee order by fname desc;--  in a to a order
	
-Group by: to summarize the data by given category 
===============================
select gender, count(gender) from employee group by gender;


-Aggregiate function 
=============================
count()		: return value count
	select count(*) from <tblname> ; --return row count
	select count(price) from <tblname>; --return value count, null value will be discarded

max()		: return highest value
min()		: return lowest value
sum()		: return total
avg()		: return mean 

etc.


-Having : filter the row after group by 
===========================================
-having can be used only with group by
